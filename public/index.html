快速开始 / Quick Start

1. 部署 Server（你只需要一个 node server.js）
$ curl -fsSL https://cloudsh.yxpil.com/install.sh | bash复制
2. 注册获取凭证 / Register
$ cloudsh -s http://YOUR_VPS:3000 register复制
3. 被控主机启动 Agent
$ curl -fsSL https://cloudsh.yxpil.com/bin/cloudsh-agent-darwin-arm64 -o cloudsh-agent
$ chmod +x cloudsh-agent
$ CLOUDSH_SERVER=http://YOUR_VPS:3000 \
  CLOUDSH_CLIENT_ID=xxx \
  CLOUDSH_PASSWORD=*** \
  ./cloudsh-agent复制
4. 执行命令 / Exec
$ cloudsh -s http://YOUR_VPS:3000 exec whoami复制
$ cloudsh -s http://YOUR_VPS:3000 exec "df -h"

一键安装 / One-Line Install

$ curl -fsSL https://cloudsh.yxpil.com/install.sh | bash复制

自动安装 Rust → 下载源码 → 编译 → systemd 服务。或指定组件：... | bash -s server

安全 / Security

纯文本转发
拒绝二进制和控制字符（\x00 \x1b），只过纯文本命令
常量时间密码比较
timingSafeEqual 防时序攻击
无 SSH 凭证存储
Server 不存目标机密码，Agent 本地执行
